====== 02 - Crontech Platform ======
Criado sexta 24 novembro 2023

Novo projeto para criação de uma plataforma Crontech de aplicações de TI baseado em:

* Infraestrutura Kubernetes
* Frontend Baseado no Vue 3 + Nuxt 3 + Prime + Template Base com Layer
* Low Code com Node Red
* RPA com OpenRPA e Puppeter
	* Gestor de Orquestração integrado com a plataforma
* Banco de dados Vitess
* Backend em Nodejs com aplicação micro e Fastfy
* Autenticação com Keycloak
* IA com as melhores ferramentas do mercado para integrar tudo
* Multi Tenent por padrão
* Integraćão com hardware node-red

* Criar um login unico que consegue acessar todas as aplicações
* Possibilidade de configurar aplicações de forma independente (exigencia do cliente)
* Criar o maximo de serviços/aplicações para revenda
