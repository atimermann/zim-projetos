====== Tarefas - 2016-11-28 ======
Created segunda 28 novembro 2016


Leonardo pediu que em caso de erro na automação do processo de upload, seja tomada as seguintes ações:
- Caso o erro possa ser resolvido com uma ação do usuário, basta apenas indicar com uma mensagem na tela
- Caso o erro não possa ser resolvido com uma ação do usuário, então a extensão deverá enviar a seguinte mensagem de erro:
		 "Eprot não está disponível para este sistema, entre em contato com o suporte"
   **Em seguida bloquear a extensão para o usuário (bloqueio não está definido no escopo do projeto)**


- Leonardo pediu também uma lista com todas as mensagens de erro e quando elas ocorrem, em especial para mensagem exibida quando o usuário se desconecta
