====== Guardian ======

* Existe integração via [[WebService]] (Documentação será enviada pelo Guilherme)  (Parece ser SOAP)
* Desenvolvedor do Guardian é o Toledo (fabricante da balança) 
* Não tem acesso a base de dados 
* Se mexer o banco perde garantia
* Não existe um servidor de teste/Homologação (entrar em contato com a Toledo para verificar esta possibilidade)
* Guilherme disponibilizou documentação do WebService
* Não existe licença para execução pelos robôs, talvez seja necessário comprar outra licença
* Unisolution já vez uma integração com o Guardian, podemos entrar em contato para verificar como
* toledobrasil.com.br
