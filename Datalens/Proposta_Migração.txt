====== Proposta Migração ======
Criado quinta 21 dezembro 2023

Proposta Comercial para Serviços de Migração

===== Introdução =====

Este documento apresenta a proposta para  migração de sistema de automação Datatem, incluindo a preparação de ambiente, migração de dados, implantação de sistema de automação e criação de documentação de arquitetura.

===== Requisitos Técnicos =====

- **Banco de Dados:** SQL Server
- **Ambiente de Operação:** Máquina virtual Linux, (Debian ou Ubuntu), compatível com Docker, com acesso ao banco de dados SQL Server.

===== Detalhes do Projeto =====

==== Preparação de Ambiente ====

- **Atividades:**
  - Atualização do sistema
  - Instalação e configuração de dependencias do serviço de automação.

**Prazo:** 4 horas

==== Migração de Dados para Nova Plataforma ====
- **Atividades:**
  - Migração dados do banco de dados para novo  servidor

**Prazo:** 4 horas

==== Implantação da Última Versão do Sistema de Automação ====
- **Objetivo:** Implementar a última versão do sistema de automação, configurá-lo para o ambiente do cliente e realizar testes.
- **Atividades:**
  - Instalação e configuração do sistema.  
  - Testes rigorosos para garantir funcionamento adequado.

**Prazo:** 16 horas

==== Criação de Documentação de Arquitetura ====
- **Objetivo:** Fornecer uma compreensão clara da nova arquitetura do sistema.
- **Atividades:**
  - Documentação técnica detalhada.
  - Diagramas de arquitetura e fluxos de processo.

**Prazo:** 16 horas

**Total:** 40 horas
