====== 02 - Uso Básico ======
Criado quinta 02 março 2023

Paperless é uma aplicação para gerenciar documentos pessoais escaneados, que consiste em duas partes: 

* um consumidor que adiciona documentos escaneados a partir de uma pasta especificada 
* um servidor web com uma interface de usuário para gerenciar e pesquisar documentos escaneados.

Cada documento tem vários campos que podem ser atribuídos a ele:

* Um **documento** é um pedaço de papel que contém informações valiosas.

* O **correspondente** de um documento é a pessoa, instituição ou empresa que originou ou recebeu o documento.

* Uma **tag** é um **rótulo** que pode ser atribuído a documentos. As tags permitem agrupar vários documentos em um único rótulo, e um único documento pode ter várias tags.
 
* Um **tipo de documento** é usado para identificar o tipo de um documento, como carta, extrato bancário, fatura, contrato, etc.

* A **data de adição** de um documento é a data em que o documento foi digitalizado para o Paperless. Essa data não pode e não deve ser alterada.

* A **data de criação** de um documento é a data em que o documento foi originalmente emitido. Isso pode ser a data em que você comprou um produto, assinou um contrato ou a data em que uma carta foi enviada para você.

* O **número de série do arquivo (ASN)** de um documento é o identificador do documento em seus arquivos físicos. Veja o fluxo de trabalho recomendado abaixo.

* O **conteúdo de um documento** é o texto que foi reconhecido por OCR a partir do documento. Esse texto é inserido no mecanismo de busca e é usado para combinar tags, correspondentes e tipos de documentos.
