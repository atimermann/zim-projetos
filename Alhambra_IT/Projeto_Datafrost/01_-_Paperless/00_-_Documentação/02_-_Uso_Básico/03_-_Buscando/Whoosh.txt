====== Whoosh ======
Criado quinta 02 março 2023

Whoosh é uma biblioteca Python que ajuda a criar e pesquisar índices de texto completo. 
Ela é usada pelo Paperless para criar e pesquisar índices de documentos digitalizados, permitindo que os usuários encontrem facilmente documentos por meio de pesquisa de texto completo. 
Whoosh é uma biblioteca de código aberto e gratuita.
